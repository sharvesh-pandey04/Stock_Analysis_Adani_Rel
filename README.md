# 📊 Stock Relation Analysis Project

## 📌 Overview
This project analyzes stock market data using Python.

We compare two stocks:
- Reliance Industries
- Adani Power

The goal is to find the relationship between them using data analysis.

---

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib

---

## 📂 Dataset
- reliance_stock.csv
- adani_power_stock.csv

---

## 📊 What I Did
- Loaded stock data using Pandas  
- Cleaned and processed the data  
- Calculated:
  - Mean (Average)
  - Median
  - Standard Deviation  
- Found **correlation (relation)** between stocks  
- Visualized data using graphs
